SAtHI-AI — Your Personalized Training Arc

SAtHI-AI is a gamified learning roadmap generator that creates a personalized training journey for developers.
SAtHI-AI generates a structured learning arc.

The idea is inspired by RPG progression systems, where each developer becomes a player completing quests to defeat their Final Boss (Goal).

-- Features
 --Personalized Learning Path

-Users choose the programming language or technology they want to master.
-Training Arc Duration
-Set how long the learning journey should last (weeks/months).
-Daily XP Grind
-Choose how much time or effort you can dedicate daily.
-Final Boss Selection
-Define the ultimate goal such as:
-Cracking FAANG interviews
-Building a full-stack product
-Mastering Data Structures & Algorithms
-Becoming a backend engineer

-- Smart Roadmap Generation

Based on user inputs, the system generates a structured roadmap.
QuestForge UI allows users to input their learning parameters and generate a training roadmap.

Example Inputs:

Language → JavaScript
Duration → 3 Months
Daily XP → 3 Hours
Final Boss → Full Stack Developer

Output → Structured roadmap with progressive learning steps and the users can also download the roadmap in PPT format.
